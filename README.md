# AndroidWeekly

一个第三方 androidweekly.net 的客户端，主要用 jsoup 抓取了 androidweekly.net 的数据进行展示，很适合在空暇时间阅读浏览。

## 目前实现的功能

1. 文章列表
2. 文章页面
3. 收藏文章
4. 调整文章字体大小
5. 文章页面内建词典
6. 文章分享
7. 已读文章全文搜索
8. 收藏导入导出

欢迎提 pull request 或 issue 添加新功能，修 bug.
